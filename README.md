# UserManagementSystem

A Full Stack web application built with React.js and .NET Web API.

## Tech Stack
- Frontend: React.js
- Backend: .NET Web API (C#)
- Database: SQLite
- ORM: Entity Framework Core

## Features
- Add new user (Name, Address, Salary)
- View all users in a table
- Edit user with modal dialog
- Delete user with confirmation dialog
- Form validation
- REST API with GET, POST, PUT, DELETE

## How to Run

### Backend
cd UserManagementAPI
dotnet run

### Frontend
cd user-management
npm install
npm start
